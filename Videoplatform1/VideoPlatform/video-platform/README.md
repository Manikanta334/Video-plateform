A full stack application built using Node.js, Express, React, MongoDB, JWT authentication and AWS S3.

- Secure JWT authorization

- User-specific video listing
- Delete videos

## Tech Stack
Frontend: React  
Backend: Node.js, Express  
Database: MongoDB  
Storage: AWS S3  

## How to Run
Backend:
cd backend
npm install
node server.js

Frontend:
cd frontend
npm install
npm run dev
